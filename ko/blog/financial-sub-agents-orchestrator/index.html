다각도 금융 의사결정의 필요성

단순히 재무제표만 보거나 단발성 뉴스 소식만 믿고 투자하면 리스크를 분산하기 어렵습니다. 각 분야 전문가 에이전트를 모아 병렬 회의를 시켜야 합니다.

서브에이전트 오케스트레이션

재무제표를 스캔하는 재무 분석 에이전트, 실시간 헤드라인 뉴스의 감정을 분류하는 뉴스 감성 분석 에이전트, 반도체 및 OSAT 동향을 추적하는 업종 리서치 에이전트를 가동합니다. 이 종합 보고서들을 취합하여 최종 공격적 전략 에이전트가 실제 계좌의 비중 차이를 상쇄하기 위한 매수 추천 수량을 정교하게 연산해 냅니다.

연관도 계산 방식 알아보기

본 포트폴리오의 추천 콘텐츠는 Google Gemini API(gemini-embedding-2 모델)를 활용해 추출한 다차원 텍스트 임베딩 벡터 간의 코사인 유사도(Cosine Similarity)를 기준으로 정렬됩니다.

글의 제목, 카테고리, 요약 설명 및 전체 본문을 종합 분석하여 단순한 키워드 일치를 넘어선 의미론적 연관성(Semantic Similarity)을 판별합니다. API 호출이 불가한 환경에서는 로컬 TF-IDF 기반의 단어 빈도 분석 알고리즘으로 자동 전환(Fallback)되어 연관도를 계산합니다.
