Vibe Coding의 한계점

단순히 AI 채팅창에 "코드 짜줘"라고 입력하는 방식은 소규모 스크립트 작성에는 유용하나, 엔터프라이즈급 프로젝트나 복잡한 시스템 리팩토링에서는 반드시 실패합니다. AI가 문맥을 잃거나 기존의 다른 모듈을 망가뜨리기 때문입니다.

Harness Engineering이란 무엇인가?

하네스 엔지니어링은 AI 에이전트가 동작하는 런타임 환경에 물리적인 통제망과 규격을 세우는 것을 의미합니다. 에이전트의 파일 접근 범위를 제한하고, 에이전트가 호출할 수 있는 도구의 가짓수를 줄이며, 사전에 정의된 규칙 가이드에 따라서만 동작하도록 제약 조건을 설계하는 것이 핵심입니다.

연관도 계산 방식 알아보기

본 포트폴리오의 추천 콘텐츠는 Google Gemini API(gemini-embedding-2 모델)를 활용해 추출한 다차원 텍스트 임베딩 벡터 간의 코사인 유사도(Cosine Similarity)를 기준으로 정렬됩니다.

글의 제목, 카테고리, 요약 설명 및 전체 본문을 종합 분석하여 단순한 키워드 일치를 넘어선 의미론적 연관성(Semantic Similarity)을 판별합니다. API 호출이 불가한 환경에서는 로컬 TF-IDF 기반의 단어 빈도 분석 알고리즘으로 자동 전환(Fallback)되어 연관도를 계산합니다.
