How similarity is calculated

Recommended content on this portfolio is sorted based on the Cosine Similarity between multi-dimensional text embedding vectors extracted using the Google Gemini API (gemini-embedding-2 model).

By analyzing the title, category, description, and full body text, it evaluates semantic relevance beyond simple keyword matching. If the API is unavailable, it automatically falls back to a local TF-IDF term frequency analysis algorithm.
